Описание:

Главный герой Джек Рурк был атакован мафией и вынужден участвовать в гонке под названием «The Run», начинающуюся в Сан-Франциско и заканчивающуюся в Нью-Йорке, чтобы спасти свою жизнь. Need for Speed: The Run сочетает в себе уличные гонки с полицейскими преследованиями. Помимо этого, в игре присутствуют интерактивные действия, QTE, с помощью которых игрок продвигается по сюжету.
