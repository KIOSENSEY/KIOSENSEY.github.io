Описание:

Версия игры для PlayStation 3 включает в себя несколько загружаемых бонусов по сравнению с версией для Xbox 360. В улучшенном издании GoldenEye 007: Reloaded предлагает улучшенные визуальные эффекты высокой четкости, работающие со скоростью 60 кадров в секунду. Заметен значительный прогресс в качестве графики. Также создатели реализовали набор графических эффектов, которые не поддерживаются менее мощными игровыми системами. Версия для PS3 поддерживает дополнительный контроллер, то есть PlayStation Move опционально расширен до Sharp Shooter. Однако они оба не обязательны, поэтому игрок может наслаждаться игрой с геймпадом в руках.
