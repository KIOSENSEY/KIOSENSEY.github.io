Описание:

Stacking - это приключенческая игра, действие которой происходит в эпоху депрессии 1920-х годов, в которой игрок играет Чарли Блэкмора, русской куклы, чью семью похитил промышленник, известный под именем «Барон», который похищал людей в своих гнусных целях. Для прохождения каждого уровня Чарли должен складываться в дополнительных русских кукол, чтобы решить несколько головоломок на каждом уровне с несколькими решениями. Помимо прохождения каждого из уровней, на каждом из них есть бонусные цели, которые игроки должны решить (привет-шутки).
